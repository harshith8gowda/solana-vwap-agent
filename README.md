# Solana VWAP Agent

## Overview
The Solana VWAP (Volume Weighted Average Price) Agent is a sophisticated trading bot that operates on the Solana blockchain. This project aims to facilitate effective trading strategies by calculating the VWAP, which is essential for assessing the average price at which a security has traded throughout a given time period, adjusted for volume.

## Features
- **Real-time VWAP Calculation**: The agent computes the VWAP in real-time by integrating data from Solana's on-chain transactions.
- **Efficient Algorithm**: Utilizes optimized algorithms to ensure timely execution of trades based on VWAP signals.
- **Custom Configurations**: Users can configure various parameters such as time intervals and trading strategies to align with their goals.
- **Comprehensive Analytics**: Provides detailed analytics and reporting features to better understand the trading performance and market trends.

## Installation
To install the Solana VWAP Agent, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/harshith8gowda/solana-vwap-agent.git
   ```
2. Navigate to the project directory:
   ```bash
   cd solana-vwap-agent
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
1. Configure the bot settings in the `config.json` file. 
2. Run the bot:
   ```bash
   npm start
   ```
